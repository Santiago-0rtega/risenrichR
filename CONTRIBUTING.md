# Contributing to ris-enrich

Welcome! We would love your help to make `ris-enrich` better. 

## How to Contribute
1. **Report Bugs:** If you find a bug or an API that has changed, please open an Issue on GitHub.
2. **Suggest Features:** Have an idea for a new database fallback? Open an Issue to discuss it.
3. **Submit Pull Requests:** - Fork the repository.
   - Create a new branch (`git checkout -b feature/new-database`).
   - Make your changes and ensure tests pass by running `pytest`.
   - Submit a Pull Request describing your changes.

Thank you for supporting Open Science!
